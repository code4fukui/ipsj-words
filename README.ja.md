# ipsj-words

情報処理学会の情報科全教科書用語リストを提供するリポジトリです。CSVファイルとサンプルアプリケーションを含んでいます。

## データ・API

- [ipsj-words-ver.1.0_2024-4-10.csv](ipsj-words-ver.1.0_2024-4-10.csv)
- 出典: CC 4.0 BY-SA [情報処理学会 情報入試委員会 - 情報科全教科書用語リスト](https://sites.google.com/a.ipsj.or.jp/ipsjjn/wordlist)

## サンプルアプリ

[IT辞書](https://code4fukui.github.io/ipsj-words/) - 情報科の用語を検索・閲覧できるウェブアプリケーションです。
