# ipsj-words

A collection of Japanese IT and computer science terms and definitions from the Information Processing Society of Japan (IPSJ).

## Demo
[IT Dictionary](https://code4fukui.github.io/ipsj-words/) - A sample application that allows searching and browsing the term list.

## Data
The project contains a CSV file `ipsj-words-ver.1.0_2024-4-10.csv` with the following fields:

- 用語 (Term)
- 説明 (Definition)

The data is licensed under CC 4.0 BY-SA from the [IPSJ Information Entrance Examination Committee - Information Science Textbook Terminology List](https://sites.google.com/a.ipsj.or.jp/ipsjjn/wordlist).
